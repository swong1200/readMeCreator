# Eat-Da-Burger
[![License: Unlicense](https://img.shields.io/badge/license-Unlicense-blue.svg)](http://unlicense.org/)
## Description
This is a simple app that allows you to create burgers.  Once you have created them, you have the option to eat them, which will then move them from left to right.
